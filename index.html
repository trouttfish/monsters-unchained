PATCH NOTES

Monsters Unchained Rotation Builder — by Alex Troutt

v2.0 — Daily Assignment System
  • Phase 2 launched: Daily Assignment tool assigns real workers to rotation slots each day
  • History-aware algorithm prevents workers from repeating the same position or rotation within a week
  • YGOR/FRITZ slots filled by fairness queue — fewest total assignments first, oldest last as tiebreaker
  • CSV roster import parses Universal schedule exports and classifies workers by shift type automatically
  • Manual swap UI lets captains reassign any slot before confirming
  • All assignments saved to Supabase — full history tab for audit and review
  • Fake Roster Generator added for testing without a real schedule CSV
  • Export button added to Phase 1 — downloads rotation board as JSON for Phase 2 import
  • Site-wide navigation added — all three tools link to each other from every page
v1.5 — UI Polish & Quality of Life
  • Auto-retry: Generate button now retries up to 50 times automatically — no more hammering the button
  • Feedback button added — submit suggestions or bug reports directly from the tool
  • Patch Notes modal added with full version history
  • Rules Doc button links directly to the technical rules document
  • Button layout reorganized — Generate prominent on top, utility buttons clustered below
  • Greeter rotation sort fixed — endcap now always appears last as intended
  • Yapping soft rule enforced in algorithm — positions naturally avoid sharing rotations
  • Naming standardized to "Monsters Unchained Rotation Builder" across all surfaces
  • Version numbers removed from visible UI
v1.4 — Nathan's Operational Updates
  • SECONDARY reclassified as non-walking turntable — can now appear in dispatch rotations
  • Yapping Family soft rule added: LOAD, PRESHOW LOAD, GROUPER, LOCKER LOAD 1, LOCKER LOAD 2 avoid sharing rotations
  • Warning banner fires after generation if yapping positions end up together
  • RCS and RSS are now always 4-spot rotations regardless of dispatch checkbox state
  • Auto-retry: Generate button now automatically retries up to 50 times without manual re-clicking
v1.3 — Dispatch Checkbox Overhaul
  • Full dispatch checkbox system implemented (RCS box, RSS box, both boxes)
  • Relaxed mode: when boxes checked, overflow rules loosen to allow optionals in 3-spot rotations
  • Post-build optional swap: optionals move from overflow into dispatch 3-spots, then greeter 4-spots
  • For 2-box mode: additionally swaps optionals from overflow into greeter 4-spots lacking one
  • Force-merge and steal logic added to handle leftover positions in tight math scenarios
  • Post-build rebalancing: oversized rotations donate positions to undersized ones
v1.2 — Optional Position System
  • 5 true optional positions identified: SINGLES LOCKERS, STAIRCASE ROAMER, MERGE, PULSE, UNLOAD 2
  • Optionals pre-assigned to 4-spot rotations so dropping one daily gracefully makes it 3-spot
  • No boxes: RCS and RSS always get optionals; remaining 3 go to greeters and dispatch
  • Optionals never placed in RCS/RSS when dispatch boxes are checked
  • Max 1 optional per rotation enforced everywhere
v1.1 — Core Algorithm & Constraints
  • SECONDARY greeterOnly rule: randomly assigned to one of three greeter rotations each week
  • Turntable cluster: max 2 per rotation, one must be non-walking if 2 present
  • Front-room cluster: max 1 of SINGLES LOCKERS, VALIDATION, STAIRCASE ROAMER per rotation
  • Full family rule enforcement across all 8 families
  • Greeter rotations descending spatial flow, dispatch/overflow ascending
  • YGOR fixed special rotation: never shuffled, always YGOR→FRITZ→ESCORT→REGULATORY→ELEVATOR
  • Dispatch endcaps always close their rotation
v1.0 — Initial Launch
  • Web-based rotation board generator for Monsters Unchained attraction
  • 35 positions, 8 rotation types, constraint-based random algorithm
  • Generation history tracks last 8 boards to prevent repeats
  • Position pool sidebar shows placed vs unplaced positions
  • Stats bar: total rotations, 4-spot count, 3-spot count, positions placed, generation number
  • Color-coded rotation cards: anchor, greeter endcap, dispatch endcap, fixed special, optional
